Persi

Persi is a portable C library in persi/: a file-backed store with a SQL-oriented API (persi_exec_sql) plus typed table helpers. It is built as a static library and exercised by persi_demo (persi/demo.c).

The peer-discovery binary 10kgp does not link Persi today; Persi is an independent component in the same repository.

Column types

  • PERSI_TYPE_INT
  • PERSI_TYPE_FLOAT
  • PERSI_TYPE_STRING

Limits

MacroValue
PERSI_MAX_COLS16
PERSI_MAX_TABLES64

Use persi_strerror(int code) for a human-readable message.

Main functions (persi.h)

  • persi_open, persi_close, persi_sync — database file lifecycle
  • persi_create_table — define columns (persi_coldef_t)
  • persi_insert — insert row; optional out_row_id
  • persi_scan_table, persi_scan_where_eq — iterate rows via persi_row_fn
  • persi_update_row, persi_update_where_eq
  • persi_delete_row, persi_delete_where_eq
  • persi_exec_sql — execute SQL; optional persi_result_t for selects
  • persi_result_free — free query result

For concrete usage, read persi/demo.c next to the headers and implementation split (persi_core.c, persi_sql.c, persi_format.c, persi_value.c).
