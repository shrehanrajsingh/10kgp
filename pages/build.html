Build

Requirements

  • C11-compatible compiler (GCC, Clang, or MSVC)
  • CMake 3.10 or later
  • Linux or macOS: POSIX-like environment
  • Windows: Visual Studio 2019+ or MinGW-w64 with Winsock2; CMake links ws2_32 and iphlpapi for 10kgp

Unix (Linux / macOS)

git clone <repository-url>
cd 10kgp
mkdir build && cd build
cmake ..
make

Outputs:

  • build/10kgp — peer discovery CLI
  • build/persi_demo — Persi demo binary

Windows

From a Developer Command Prompt or any shell where cmake and your compiler are on PATH:

cd 10kgp
mkdir build && cd build
cmake -G "Ninja" ..
cmake --build .

Use a Visual Studio generator if you prefer, for example:

cmake -G "Visual Studio 17 2022" -A x64 ..

Run 10kgp.exe from the build directory. Networking uses Winsock2 via emit/win_emit.c; console input uses win_stdin.c so stdin is not mixed with Winsock select.

Debug build

Unix-style generators:

cmake -DCMAKE_BUILD_TYPE=Debug ..
make

On Windows with Ninja or MSBuild, pass -DCMAKE_BUILD_TYPE=Debug at configure time if your generator respects it.
