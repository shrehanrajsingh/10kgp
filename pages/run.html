Run

./build/10kgp

On launch you are prompted for a peer ID — a short identifier (up to 31 characters) that other peers see. Examples: username, hostname, or any unique label.

=== Peer Discovery ===

Enter your peer ID: alice

After entering your ID, the program sends an initial discovery pulse and listens for others. The > prompt accepts commands.

Commands

CommandDescription
/onlineList all peers currently online
/peersAlias for /online
/countPrint the number of online peers
/refreshSend a discovery pulse immediately
/clearClear the terminal screen
/alias <command> <alias>Create a shorthand for a command
/show-aliasList all defined aliases
/clear-alias <alias>Remove a specific alias
/helpShow the help message
/exitExit the application
/quitAlias for /exit

Aliases

Aliases bind a shorter name to a built-in command for the session only; they are not saved between runs.

> /alias online o
Alias '/o' -> '/online' created.

> /o
Online peers (2):
...

> /show-alias
Aliases (1):
  Alias            -> Command
  ----------------    -------
  /o               -> /online

> /clear-alias o
Alias '/o' cleared.
