10KGP

An internal peer-to-peer interaction framework for computers on a shared local network. The application runs in the terminal, uses UDP for peer discovery, and is designed to be extended with additional communication modules over time.

Project repository on GitHub (replace with your fork URL if different).

What it does

  • Discovers other instances on the same LAN (Wi‑Fi or wired).
  • Tracks peer IDs and addresses with timeouts.
  • Provides a simple slash-command CLI (/online, /refresh, etc.).
  • Includes the emit discovery library and the separate Persi file-backed store library.

Hosting this site (GitHub Pages)

The HTML lives under pages/ in the repo. GitHub’s branch settings only publish from the repo root or docs/, so this project uses a GitHub Actions workflow (.github/workflows/pages.yml) that uploads the pages/ folder as the Pages artifact.

In the repository: Settings → Pages → Build and deployment → Source: GitHub Actions. Pushes to main deploy the site.
